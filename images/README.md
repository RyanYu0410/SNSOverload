## Recommended Image Specs

- **Aspect Ratio**: 1:1 (square) works best
- **Size**: 400x400 to 800x800 pixels
- **Format**: PNG, JPG, or WebP

## Image Ideas

Since this is an art piece about late-night scrolling, consider images that evoke:

- Night sky / moon / stars
- Glowing screens in the dark
- Cozy bedroom scenes
- Abstract gradient patterns
- Lo-fi aesthetic imagery
- Soft, dreamy visuals

You can also use placeholder services for testing:

```
https://picsum.photos/400/400?random=1
https://picsum.photos/400/400?random=2
...
```

Just update the `fakePosts` array in `script.js` to use these URLs.
